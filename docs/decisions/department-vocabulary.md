# Department Vocabulary

**Decision date:** 2026-07-15

Alexandria is a DCISM thesis repository. The only permitted department values
for new submissions and later corrections are:

- `CS`
- `IT`
- `IS`

Legacy studies with other department values will be removed manually in
Supabase. This decision does not include an automated data migration.

The upload flow, correction workspaces, admin metadata corrections, review
filters, service validation, and SQL RPCs must all use this same vocabulary.
